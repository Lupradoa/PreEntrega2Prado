¿Sabés que es la impresión 3D?

La impresión 3D es un proceso de fabricación que crea un objeto físico a partir del archivo de un modelo digital. El proceso de imprimir en 3D consiste en crear los objetos por capas superpuestas de abajo a arriba. Antes de empezar la impresión, el software divide el gráfico 3D en capas tan finas como el diámetro del plástico de salida. La impresora alcanza determinada temperatura, la necesaria para calentar y derretir el material con el que se va a trabajar y para cada capa, va desplazándose sobre el plano para soltar el plástico sobre las coordenadas adecuadas. Formando finalmente una figura en tres dimensiones, un objeto que puedes tocar y usar en la vida real.

Un modelo digital en 3D se corta en cientos de capas finas mediante un software específico para exportarlo en formato de archivo .gcode, este archivo es creado en un lenguaje que la impresora 3D lee para saber con precisión cuándo y dónde depositar el material.

¿Que materiales se pueden utilizar para crear una impresión 3D?

Se puede utilizar distintos materiales para crear los objetos, los más comunes son de base plástica como ser PLA, ABS, PETG entre otros los cuales se encuentran en forma de filamentos de colores

¿Que se puede crear en una impresora 3D?

Se puede crear una amplia variedad de objetos tridimensionales, pueden ser piezas de repuestos, jueguetes, joyas, figuras, artículos de decoración, incluso también se utiliza para crear prótesis personalizadas, dispositivos de asistencia y modelos anatómicos.
